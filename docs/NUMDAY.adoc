
== NUMDAY: 通し日数の算出

=== 目的

M月N日の通し日数（1月1日を1として順に数えた日数）を算出する。

=== 入出力

.入力
[options="header" cols="1,4,2"]
|=================================
|変数名|説明|単位
|M | ⽉| 
|N | ⽇|  
|=================================

.出力
[options="header" cols="1,4,2"]
|=================================
|変数名|説明|単位
|stem:[N_{day}]| 通し日数 |
|=================================

=== 使用する関数

なし

=== 計算ロジック

====
[stem]
++++++++++++++++++++++++++++++++++++++++++++
N_{day} = 30*(M-1) + 0.5*(M+M/8) - 0.1*(M+7) + N
++++++++++++++++++++++++++++++++++++++++++++
====
