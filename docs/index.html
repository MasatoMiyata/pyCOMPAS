4. SOPOS: 太陽位置

4.1. 目的

SOPOSは、緯度、経度、⽉、⽇、時刻をもとに太陽位置を求めるプログラムである。
太陽位置を表すものとして、 sinh、sinA、cosA（h：太陽⾼度、A：太陽⽅位⾓）の3つを求める。

4.3. 使用する関数

  • NUMDAY

4.4. 計算ロジック

時角 \(t\) [rad]を次式で算出する。

通し日の算出

\[N_{day} = \rm{NUMDAY}(M, N)\]

均時差の算出

\[e = \rm{EQT}(N_{day})\]

時角[rad]の算出

\[t = (15 * (HJ - 12 + e) + LO + LOs) * \rm{RADI}\]

日赤緯[rad]を算出する。

\[\delta = \rm{DEL}(N_{day}) * \rm{RADI}\]

緯度をradに変換する。

\[\varphi = \rm{LA} * \rm{RADI}\]

太陽高度をh、太陽方位角をAとすると、太陽位置を表すsinh、sinA、cosAは次式で算出される。

\[\sin h = \sin \varphi \cdot \sin \delta + \cos \varphi \cdot \cos \delta \cdot \cos t \\ \cos h = \sqrt{1-(\sin h)^{2}} \\ \sin A = \frac{\sin t \cdot \cos \delta}{cosh} \\ \cos A = \frac{\sin h \cdot \sin \varphi - \sin \delta}{\cos h \cdot \cos \varphi}\]

ただし、sinh<0 のときは、sinh=0、sinA=0、cosA=0 とする。

4.5. 注記

  • 時刻 HJ は、⽇本標準時となっている。ELS=135.0 は明⽯の東経を⽰しているので、外国の時刻のときに は注意を要す。

  • ⽇出、⽇没を求めるプログラムにはなっていない。

  • ⽇⾚緯、均時差などは、ある⼀⽇では同⼀の値をとるので、時刻別に幾度もCALLするときなどは、DEL(W)、 EQT(W)の計算は無駄になっている。

  • W の計算において、1 年を 366 ⽇とみなしているので、最⼤ 1 ⽇分の太陽位置の誤差が出る。

5. PSY: 空気状態値の算出

5.4. 計算ロジック

5.4.1. サブ関数の定義

計算に使用するサブ関数を次式で定義する。

  • 水蒸気分圧 \(P\)[mmHg] から 絶対湿度 \(X\)[kg/kg] を算出する関数AH

\[X = \rm{AH(P)} = \frac{0.622 P}{760-P}\]

  • 水蒸気分圧 \(P_{atm}\)[atm] から 絶対湿度 \(X\) [kg/kg] を算出する関数A

\[X = \rm{A(P_{atm})} = \frac{0.622 P_{atm}}{1-P_{atm}}\]

  • 絶対湿度 \(X\)[kg/kg] から 水蒸気分圧 \(P\)[mmHg] を算出する関数CH

\[P = \rm{CH(X)} = \frac{760 X}{0.622 + X}\]

  • 絶対湿度 \(X\)[kg/kg] から 水蒸気分圧 \(P_{atm}\)[atm] を算出する関数C

\[P_{atm} = \rm{C(X)} = \frac{X}{0.622 + X}\]

  • 湿球温度\(T_{wb}\)[℃]のときの飽和絶対湿度\(X_{s}\)[kg/kg]、乾球温度\(T_{db}\)[℃]、湿球温度\(T_{wb}\)[℃]から、絶対湿度 \(X\)[kg/kg] を算出する関数B

\[X = \rm{B(X_{wb}, T_{db}, T_{wb})} = \frac{ (597.3 - 0.559T_{wb}) X_{s} - 0.240 (T_{db}-T_{wb}) }{ 597.3 + 0.441 T_{db} - T_{wb} }\]

  • 絶対湿度 \(X\)[kg/kg] から露点温度\(T_{dew}\)を算出する関数TD

\[T_{dew} = \rm{TD(X)} = 90.93743 + log(X)(16.98006 + \\ log(X)(-0.7905916 + log(X)(-0.2411693-0.01389958log(X)))) + \\ 0.8045291 | log(X)+5.580150 |\]

  • 乾球温度[℃]と絶対湿度[kg/kg]からエンタルピー[kcal/kg]を算出する関数HF

\[H = \rm{HF(T_{db}, X)} = 0.240 T_{db} + (597.3+0.441T_{db})X\]

  • 乾球温度[℃]とエンタルピー[kcal/kg]から絶対湿度[kg/kg]を算出する関数XF

\[X = \rm{XF(H, T_{db})} = \frac{H-0.240T_{db}}{597.3+0.441T_{db}}\]

5.4.2. 湿球温度が入力された場合

乾球温度 \(T_{db}\) と湿球温度 \(T_{wb}\) が入力された場合は、次式で出力を算出する。

湿球温度 \(T_{wb}\) のときの飽和水蒸気分圧 \(P_{s}\)[atm]

\[P_{s} = \rm{PS(T_{wb})}\]

湿球温度 \(T_{wb}\) のときの飽和絶対湿度 \(X_{s}\) [kg/kg]

\[X_{s} = \rm{A(P_{s})}\]

絶対湿度 \(X\) [kg/kg]

\[X = \rm{B(X_{s},T_{db},T_{wb})}\]

露点温度 \(T_{dew}\) [℃]

\[T_{dew} = \rm{TD(X)}\]

相対湿度 \(R\) [%]

\[R = \frac{\rm{C(X)}}{\rm{PS(T_{db})}} \times 100\]

水蒸気分圧 \(P\) [mmHg]

\[P = \rm{CH(X)}\]

エンタルピー \(H\) [kcal/kg]

\[H = \rm{HF(T_{db}, X)}\]

5.4.3. 露点温度が入力された場合

乾球温度 \(T_{db}\) と露点温度 \(T_{dew}\) が入力された場合は、次式で出力を算出する。

露点温度 \(T_{dew}\) のときの飽和水蒸気分圧 \(P_{atm}\) [atm]

\[P_{atm} = \rm{PS(T_{dew})}\]

水蒸気分圧 \(P\) [mmHg]

\[P = P_{atm} \times 760\]

絶対湿度 \(X\) [kg/kg]

\[X = \rm{A(P_{atm})}\]

エンタルピー \(H\) [kcal/kg]

\[H = \rm{HF(T_{db},X)}\]

湿球温度 \(T_{wb}\) [℃]

\[T_{wb} = \rm{FG(H,X)}\]

相対湿度 \(R\) [%]

\[R = \frac{\rm{P_{atm}}}{\rm{PS(T_{db})}} \times 100\]

5.4.4. 相対湿度が入力された場合

乾球温度 \(T_{db}\) と相対湿度 \(R\) が入力された場合は、次式で出力を算出する。

水蒸気分圧 \(P_{atm}\) [atm]

\[P_{atm} = \frac{\rm{PS(T_{db})} R}{100}\]

水蒸気分圧 \(P\) [mmHg]

\[P = 760 P_{atm}\]

\[X = \rm{A(P_{atm})}\]

エンタルピー \(H\) [kcal/kg]

\[H = \rm{HF(T_{db},X)}\]

\[T_{wb} = \rm{FG(H,X)}\]

露点温度 \(T_{dew}\) [℃]

\[T_{dew} = \rm{TD(X)}\]

5.4.5. 絶対湿度が入力された場合

乾球温度 \(T_{db}\) と絶対湿度 \(X\) が入力された場合は、次式で出力を算出する。

エンタルピー \(H\) [kcal/kg]

\[H = \rm{HF(T_{db},X)}\]

湿球温度 \(T_{wb}\) [℃]

\[T_{wb} = \rm{FG(H,X)}\]

露点温度 \(T_{dew}\) [℃]

\[T_{dew} = \rm{TD(X)}\]

5.4.6. 水蒸気分圧[mmHg]が入力された場合

乾球温度 \(T_{db}\) と水蒸気分圧 \(P\) が入力された場合は、次式で出力を算出する。

絶対湿度 \(X\) [kg/kg]

\[X = \rm{AH(P)}\]

エンタルピー \(H\) [kcal/kg]

\[H = \rm{HF(T_{db},X)}\]

湿球温度 \(T_{wb}\) [℃]

\[T_{dew} = \rm{D(X)}\]

相対湿度 \(R\) [%]

\[R = \frac{P}{(\rm{PS(T_{db})} 760)} \times 100\]

5.4.7. エンタルピーが入力された場合

乾球温度 \(T_{db}\) とエンタルピー \(H\) が入力された場合は、次式で出力を算出する。

\[T_{dew} = \rm{TD(X)}\]

相対湿度 \(R\) [%]

\[R = \frac{\rm{C(X)}}{\rm{PS(T_{db})}} \times 100\]

水蒸気分圧 \(P\) [mmHg]

\[P = \rm{CH(X)}\]
